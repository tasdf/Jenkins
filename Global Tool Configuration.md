## 全局工具配置

- **JDK**
```
[root@jenkins ~]# which  java
/usr/bin/java
```
![图片.png](./images/jdk_tool.png)

- **MAVEN**
```
[root@jenkins ~]# which  mvn

```
![图片.png](./images/maven_tool.png)

- **Git**
```
[root@jenkins ~]# yum install -y git
```

> Note： The paths of all machines need to be consistent
